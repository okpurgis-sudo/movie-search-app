# Movie Search App

C# WinForms と SQLite で作成した、個人用の作品データベースアプリです。

作品名、出演者名、ジャンル、ナイスポイント、評価、URLをローカルDBに保存し、検索・編集・削除できます。

## 概要

Movie Search App は、自分が見た作品や気になる作品を管理するためのローカルアプリです。

一般的な作品レビューサイトや配信サービスの検索ではなく、
自分で登録した作品データをもとに、出演者・ジャンル・自分なりのナイスポイントから検索できます。

データは外部サーバーには送信されず、ローカルPC上のSQLiteデータベースに保存されます。

## 主な機能

* 作品データの登録
* 作品データの編集
* 作品データの削除
* 作品名による検索
* 出演者名による検索
* ジャンルによる検索
* ナイスポイントによる検索
* OR検索 / AND検索の切り替え
* カンマ区切り / 空白区切りによる複数キーワード検索
* 一覧ヘッダークリックによる昇順 / 降順ソート
* セルクリックによるテキストコピー
* 出演者・ジャンル・ナイスポイントの傾向分析
* 条件に合う作品をランダムで1件表示する「気分で選ぶ」機能
* SQLiteによるローカル保存

## 登録できる項目

* 作品名
* 出演者名
* ジャンル
* ナイスポイント
* 評価
* URL

## 検索機能

作品名、出演者名、ジャンル、ナイスポイントを使って検索できます。

複数キーワードを入力する場合は、空白またはカンマで区切れます。

例：

```text
ブラッド〇ット エドワー〇ノートン
```

```text
ブラッドピッ〇,エドワード〇ートン
```

OR検索の場合は、入力したキーワードのいずれかに一致する作品を表示します。

AND検索の場合は、入力したキーワードすべてに一致する作品を表示します。

## 気分で選ぶ機能

出演者名、ジャンル、ナイスポイントのいずれかを入力して「気分で選ぶ」を押すと、条件に合う作品の中からランダムで1件表示します。

次に見る映画を自分で決めきれないときや、登録データの中から気分に近い作品を選びたいときに使えます。

## 傾向分析機能

登録済みデータから、以下の項目の出現回数を集計できます。

* 出演者名
* ジャンル
* ナイスポイント

自分がよく登録している出演者やジャンル、好みの傾向を確認できます。

## データ保存場所

登録データはSQLiteデータベースとして、以下の場所に保存されます。

```text
%LOCALAPPDATA%\MovieSearchApp\MovieSearchApp.db
```

通常のWindows環境では、以下のような場所になります。

```text
C:\Users\ユーザー名\AppData\Local\MovieSearchApp\MovieSearchApp.db
```

データをバックアップしたい場合は、この `MovieSearchApp.db` をコピーしてください。

## ダウンロード

最新版は GitHub Releases からダウンロードできます。

```text
https://github.com/あなたのユーザー名/movie-search-app/releases
```

リリースページから `MovieSearchApp_v1.0.0.zip` などの配布ファイルをダウンロードしてください。

zipファイルを展開し、同梱されているインストーラーを実行してください。

## インストール方法

1. GitHub Releases から最新版のzipファイルをダウンロードします。
2. zipファイルを展開します。
3. `MovieSearchApp_Setup_v1.0.0.exe` を実行します。
4. 画面の案内に従ってインストールします。
5. インストール後、スタートメニューまたはショートカットからアプリを起動します。

個人制作アプリのため、Windowsの警告が表示される場合があります。

## アンインストール

Windowsの「設定」→「アプリ」からアンインストールできます。

アンインストールしても、環境によっては登録データが残る場合があります。

完全に削除したい場合は、以下のフォルダを手動で削除してください。

```text
%LOCALAPPDATA%\MovieSearchApp
```
